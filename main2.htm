Entrance

前言
- 全ての始まりと、何時か辿り着く結末

這裡是放心得、生活點滴回顧的地方，說得更直截了當點，就是自己寫給自己爽的自慰文總輯。標題用「Gameの部屋」純粹只是因為在這階段我的心得最多出於（電子小說）遊戲中。假如在高中時我就有機會架站的話，那時的標題可能是「文学の部屋」、「人生軌跡」之類。ま、もう過ぎた事だ。會記得如此仔細，是因我有種習慣，喜歡將所有「出自己身」的東西——不論是一字一句——留下來作成紀錄。這可能沒什麼用，但也不至於有什麼障害就是了。
雖然裡面可能包含一些狂妄的話語，但我想這些文字還是表達了真實的我。讀了這些，也就代表認識了我——甚至會超越現實中我的工作伙伴。
可能是因為長久以來寫日記的訓練，我的文筆頗為固執，許多觀念不一定為人所包容；假如想拿來當參考的，請自己斟酌包容。それで。

舊前言

掙扎了些時候，終於還是決定談談有關 game 的心得。
其實只是因為，懶得打字。
不過，隨著經過一個又一個的 game，未記下的心得總會全淡忘了。漸漸感到可惜。
就這樣，終於以日記的形式來記述……旁邊有以我玩的順序排列及各假名的索引。英文也全轉作假名了。因有些ネタバレ，所以請稍有相關知識（最好玩過）再來看，不熟的就隨便逛逛吧。（重要的是，還有很多沒玩過就不會瞭解的部分。）

評價的分數是參考用的，其實沒啥標準 (>_<;;)。至於前十幾個 game 沒有，後來才加上「(☆)」之類的評價，由低至高是(☆)→☆→(★)→★。基本上★是主要方法（其他是微調），★★★及格（有時這上下的雖有些微差距也都被我打成★★★了），五顆星為滿分；但是可以看到許多特例……（已經不算特例了←笑）
在此之前，有句話想先說的。在臺灣，玩ビジュアルノベル實在是相當幸福的了。臺版的價錢是在日本的十分之一以下，（考慮物價，其實也差不多是八九分之一。）所以想研究的大可不必那麼在意價錢，該入手的就買吧！（★注意：這裡並不鼓勵玩這些遊戲。尤其對於一些犯法的行為與描述，假如您很容易受遊戲影響，或甚至蓄意以遊戲中的劇情實作於現實中，那還是別這麼做吧。這樣不會對您有任何益處。←念のために、一応……）

不過，假如您是真的想欣賞好內涵的話，請放棄吧。找找看純文学，像是青空文庫。（順帶一提，免費的呵！這個不用成本，不怕被人坑；而且包準到老都不會令人後悔，還喃喃自語：「俺年輕時多麼荒唐……」）裡面應該有許多比較適合您的。遊戲的內涵，以 game 來說，應該是最接近電子小說的ビジュアルノベル會最深。但據我玩到現在的感觸（虧我還專挑被評為「好」劇本的遊戲們），能與純文学相互輝映者幾稀。至今就算最深刻、最引人的劇本，碰到真正的「好」故事、「好」小說，也得退避三舍（當然不是為了禮讓）。更遑論一般ビジュアルノベル，那無異是班門弄斧。再說道一般 game，例如 RA/Age 等，那就……浪費生命的 game，讓我代替月亮消滅你吧！（啊這……從、從哪、哪哪哪裡冒出來的……）
這一方面是因為現在的玩家大多承受不住純文学的枯燥及言外之意；（想想看古典音樂與流行音樂的差距及接受度，大概就能略懂七分。）另一方面アダルトゲーム的劇本作家大多還沒具備足夠素養。這可能得從早期 198X 年代說起，那時至今的潮流（實用性為主）造就了今日的アダルトゲーム與劇本作家。簡單的說，很遺憾的，會高歌アダルトゲーム劇本萬歲、其他都是雜碎的一群人（以白吃 k氏為首←啊、我快放棄了），大多就處於這麼高不成、低不就的立場。
從這裡的文章不難發現我過度的執著於「內涵」，執著到不但有點煩，有時還令人覺得沒必要也沒意義的程度。假如要問為什麼的話，或許是因為我還存在這世上原因之一，就是為了這東西吧。請不用介意，遇到很奇特的說法……就莞爾一笑置之吧。

這邊的文章大多使用強烈批評的語調，也曾被批評過是犬儒主義（的確有某些程度的類似性……）。所以若您是某作品的死忠支持者，建議您別看為妙。
另外，雖然這麼說可能有點殘酷，不過，閱讀本站文章恐怕得具備基礎日文程度。

  2. 回溯的處理能和普通行進時一樣，不只是劇情，圖像、音聲等多媒體效果都能同時回溯。

  3. 有 YUNO 或コンチェルトノート般的流程圖（又如百鬼的地圖式索引系統）及進行階段進度表，或最起碼水夏~SUIKA~的讀過橋段選擇。

  4. 讀過的記錄完整，skip 時能選擇直接跳過整個段落（如青空及腐り姫），或以超快速度 skip；就算有還沒讀過的，最好只在碰到沒讀過的幾句才停。

  5. 對話框隨人移動，並能調整位置及大小，對話配色（如天藍の夏，每個人的話皆可以有不同色彩。並且能自訂字體等。），並能自訂色彩及字體（陰影、描邊、去除鋸齒等）。

  6. 聲光效果能多采多姿而迅速，以 2000年的現在，例如 âge 的 rUGP 系統。

  7. 美工及遊戲設計等完成度高，如 Moon Light。

  8. 音聲不照本宣科，且切換迅速（別像某些 game 一次語音停頓一次）。能重複播放。各類型音量能自由調控大小及有無。

  9. Page Up/Page Down 就可以 review，並能用上下控制 mouse 移動等等，由 keyboard 即可控制全部（最起碼是常用的）動作。

  10. 僅將最重要的功能列於面板（遊戲畫面）上，別像顔のない月還擺花在上面。

  11. 能和ゆきうた一樣，用滾輪控制文章的行進。

  12. save 檔極小。能自動於選項與各重要章節前 auto save，由 menu 選擇 continue 即可自動 load auto save。

  13. 穩定度極高。

  14. 和秋桜の空に一樣，遊戲一邊進行（如 skip）還能一邊停留在 tool bar 選擇選單之中。

  15. 劇本使用 HTML 方式，有てんしのはしご一般的字典功能，達到讀過的連結也能變色。skip 時自動取消。

  16. 滑鼠追尾（或最起碼在選択肢能跳至預設選項）及撳按鍵後滑鼠游標(cursor)自動消失（如 Fate）。

  17. 全螢幕後再回到視窗模式，其他應用程式也不跑掉。全螢幕時能切換到其他程式。

  18. 螢幕尺寸能自己調整，滑鼠及鍵盤的功能能自己設定（那個功能用什麼快速鍵）。

  19. 能統計各章節（部分/區域）與所有使用時數，並分全部與前景執行時間。

  20. 功能皆為選擇性的（所有設定皆能自由設定有無及程度）。例如能選擇是否背景處理（程式在背景時是否繼續 skip 或播放音樂等）。設定可儲存成偏好檔（如我們對於 OS 的布景主題所作的期待），並有基礎及進階（專家）模式的設定。

  21. 布景(theme)除了完全控制的自由度之外，能為懶惰的玩家們開出幾套可選擇的套裝設定。有必要再由玩家自己做各方面的微調。
